Install the LWB Bible App

iPhone / iPad (Safari)

  1. Open kjv.livingwordbibles.com in Safari.
  2. Tap the Share icon.
  3. Choose Add to Home Screen.
  4. Tap Add.

Android (Chrome)

  1. Open kjv.livingwordbibles.com in Chrome.
  2. Tap the ⋮ menu.
  3. Tap Install app (or Add to Home screen).

Desktop (Chrome / Edge)

  1. Open the site.
  2. Click the Install icon in the address bar.
